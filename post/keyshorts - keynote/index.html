Keynote 快捷操作

2020-06-08 4 min read # 学习 Learning

详细可参考Keynote演讲快捷键

通用快捷键

| 存储为 | Option-Shift-Command-S |
| 最小化窗口 | Command-M |
| 进入全屏幕视图 | Control-Command-F |

在演示文稿内四处移动

| 移到当前文本区域的开头 | Command-上箭头键 |
| 移到当前文本区域的底部 | Command-下箭头键 |
| 移到当前行的开始位置 | Command-左箭头键 |
| 移到当前行的结束位置 | Command-右箭头键 |
| 移到第一张幻灯片 | Fn-左箭头键或 Home 键 |
| 移到最后一张幻灯片 | Fn-右箭头键或 End 键 |

选择文本

| 将粗体应用到所选文本 | Command-B |
| 将斜体应用到所选文本 | Command-I |
| 将下划线应用到所选文本 | Command-U |
| 使字体大小变大 | Command-加号 (+) |
| 使字体大小变小 | Command-减号 (-) |
| 使文本左对齐 | Command-左大括号 ({) |
| 使文字居中 | Command-竖线 (|) |
| 使文本右对齐 | Command-右大括号 (}) |
| 使文本左右对齐（两端对齐） | Option-Command-竖线 (|) |

格式化文本

| 将文本转换为链接 | Command-K |
| 拷贝段落样式 | Option-Command-C |
| 粘贴段落样式 | Option-Command-V |

文本准确性

| 查找与替换 | Command-F |
| 检查拼写和语法 | Command-分号键 (😉 |
| 显示“拼写和语法”窗口 | Shift-Command-冒号键 (😃 |

导航视图

| 显示或隐藏对象列表 | Shift-Command-L |
| 成组所选对象 | Option-Command-G |
| 取消成组所选对象 | Option-Shift-Command-G |
| 将所选对象移到最后 | Shift-Command-B |
| 将所选对象后移一层 | Option-Shift-Command-B |
| 将所选对象移到最前 | Shift-Command-F |
| 将所选对象前移一层 | Option-Shift-Command-F |
| 锁定所选对象 | Command-L |
| 解锁所选对象 | Option-Command-L |
| 复制对象 | Command-D 或按住 Option 键拖移 |
| 创建与上一张所选幻灯片位于同一层次的新幻灯片 | Return 键或 Shift-Command-N |
| 向右缩进所选幻灯片 | Tab 键 |
| 向左移动缩进的幻灯片 | Shift-Tab 键 |
| 选择多张幻灯片 | 按住 Shift 键拖移 |
| 扩展或减少幻灯片选择 | 按住 Shift 键点按 |
| 复制幻灯片 | Command-D |
| 删除所选幻灯片 | Delete 键 |
| 移到下一张幻灯片 | 下箭头键 |
| 移到上一张幻灯片 | 上箭头键 |
| 展开幻灯片组 | 右箭头键 |
| 折叠幻灯片组 | 左箭头键 |
| 跳过某张幻灯片以使其不显示在演示文稿中，或显示某张被跳过的幻灯片 | Shift-Command-H |

显示 - 看片台视图

| 移到下一张幻灯片 | 右箭头键 |
| 移到上一张幻灯片 | 左箭头键 |
| 将选择扩展到下一张幻灯片 | Shift-右箭头键 |
| 将选择扩展到上一张幻灯片 | Shift-左箭头键 |
| 将选择扩展到第一张幻灯片 | Shift-Command-上箭头键 |
| 将选择扩展到最后一张幻灯片 | Shift-Command-下箭头键 |
| 选择第一张幻灯片 | Command-上箭头键 |
| 选择最后一张幻灯片 | Command-下箭头键 |

播放演示文稿并使用演讲者模式

| 播放演示文稿 | Option-Command-P |

在演示过程中控制视频

| 操作 | 快捷键 |
| 播放视频 | 空格键 |
| 暂停或继续播放视频 | K 键 |
| 倒回视频（如果已暂停播放，按帧倒回） | J 键 |
| 快进视频（如果已暂停播放，按帧快进） | L 键 |
| 跳到视频的开头 | I 键 |
| 跳到视频的结尾 | O 键 |
